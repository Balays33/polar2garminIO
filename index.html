Polar2Garmin

Make a Polar TCX file compatible with Garmin Connect

Why we do it? When importing TCX files into Garmin Connect that has been exported from Polar you need to remove above mentioned elements from the TCX file before you can successfully import the file.

How we do it? No information is saved to the server, everything is done on client side.
